# SkillCraft Technology - Data Science Task 01

## Data Visualization using Bar Chart

This project focuses on visualizing the distribution of a categorical variable using a bar chart. The visualization helps in understanding data distribution and comparing category frequencies effectively.

### Features

* Created a bar chart to visualize data distribution
* Analyzed categorical data using graphical representation
* Applied basic data visualization techniques

### Technologies Used

* Python
* Pandas
* Matplotlib

### Project Structure

```text
SCT_DS_1_DataVisualization/
│
├── data_visualization.py
├── output.png
└── README.md
```

### How to Run

1. Clone or download the repository.
2. Install the required libraries:

   ```bash
   pip install pandas matplotlib
   ```
3. Run the Python script:

   ```bash
   python data_visualization.py
   ```
